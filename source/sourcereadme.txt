To use the app, keep in mind the following:
    1. There is no provision provided in case the user forgets his password, so always remember your password
    2. The signup page asks the user for information along with a list of topics to subscribe
    3. The feed of the user displays all questions with any tagged topic being a subscribed topic for the user as well as the questions which are asked by the user
    4. The post question page can be used to ask a question
    5. The user can post answer on a question and upvote/downvote some existing answer at most once
    6. The notifications page displays whether some user has answered a question asked by you or upvoted/downvoted your answer
    7. The personal page displays the logged in user's info along with the cool metric of totalvotes gather by the user
